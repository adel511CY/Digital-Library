

# تشغيل الموقع بشكل المطلوب 

في البداية لابد تعرف انه ليس الزامي تحمل الكثر من المكتبات الامر جدا بسيط <br> اولا عندك الان موقعين واحد فيه الثغرات والاخر مغلقه الثغرات فيه 

اذا كنت تحتاج لتجربة الثغرات يمكنك تحميلها عبر ذا الامر 
--

` git clone https://github.com/adel511CY/Digital-Library/The%20APPS/web%20vulnerability `

اما بنسبة للتطبيق المحصان من الاربع ثغرات عبر ذا الكود 
--

` git clone https://github.com/adel511CY/Digital-Library/The%20APPS/web `



بعد ذي الخطوات الدخل للمجلد إما "web%20vulnerability" او "web" او كلاهم اذا حملت الاثنين عن طريق الامر `cd web%20vulnerability ` OR `cd web `

في نقطة جدا مهم وهي اعطاء بعض الصلاحيات للملفات عن طريق ذي الاوامر 

`chown -R %USER:%USER . `
<br> ` chmod -R 755 . `

بعدها نفذ الامر التالي 

` npm install `
 
والان اذا انت محمل التطيبق المحصن  يجب تتبع بعض الخطوات وهي كل الاتي 

1. انشاء ملف باسم `.env `
1. بعدها ضيف الامتغير التالي `JWTSSS="اي قيمة تعجبك" `

---
الان طريقة تشغيل الموقع كل الاتي عن طريق الامر التالي
--

` node server `

الان يمكنك تذهب الى 

`http://localhost:3000/`
